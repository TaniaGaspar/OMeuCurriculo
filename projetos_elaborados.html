Programa de envio de e-mails

Programa com vista a enviar muitos e-mails de forma automática e sem ser considerado spam, segundo uma dada lista em excel.
Este programa funciona na base do outlook, sendo por isso obrigatório ter a conta de outlook ligada com a conta de onde se pretende enviar os e-mails.
Aqui temos a interface inicial.


Aqui temos um exemplo de como se comporta quando mandamos enviar um e-mail.


Depois o segundo botão serve para enviar um e-mail de teste, para termos a certeza de como será rececionado o nosso e-mail.


Quando queremos mudar o assunto e a mensagem a enviar vamos ao botão de mudar a mensagem e podemos alterar logo aqui, contudo o corpo de texto deve estar escrito com as tag de HTML, algo que o utilizador comum não sabe fazer.


Para resolver este problema resolvi usar uma tecnologia de WYSWYG online, que podemos aceder no botão de construtor de email.


Abre esta página no browser pré-definido, algo com um aspeto identico a isto. Dá para alterar o assunto e usar diversas opções estéticas no corpo do e-mail tais como:

  • negrito
  • sublinhado
  • alterar fonte de letra
  • colocar cor de fundo
  • mudar a cor de letra
  • fazer listas ordenadas e não ordenadas
  • alterar a disposição do texto
  • inserir tabelas
  • inserir links podendo escolher o texto que é apresentado
  • inserir imagens e vídeos e redimensionar estes
  • por fim, podes ver o código que foi gerado em HTML, que à partida não vai ser preciso

Basta escrever o e-mail e guardar. Debaixo do botão de guardar está a mensagem guardada anteriormente, o que é muito utila para o caso de haver uma gralha e só se queira fazer uma pequena alteração, desta forma é só copiar e colar na caixa de texto acima.


Ainda na alteração da mensagem, podemos colocar anexos (máximo 5).
Para adicionar um anexo tem de colocar o caminho, o nome do arquivo e a extensão/tipo do ficheiro como está descrito no programa.


Após estas alterações é aconselhavel usar sempre o e-mail de testes para ver o resultado.


Passando para o botão de mostrar e-mails, aqui aparece a lista de e-mails que falta enviar, este é um exemplo de lista vazia.


E por último, mas não menos importante, as outras opções.

Aqui podemos:

  • alterar a quantidade de e-mails a enviar de cada vez (neste exemplo estamos a enviar 5).
  • definir qual será o e-mail para onde queremos enviar o e-mail de teste
  • importar a lista de e-mails original, isto porque sempre que um e-mail é enviado este é retirado da lista
  • definir se queremos esta importação de forma automática ou não
  • ver a lista de e-mails que foram rejeitados pelo outlook (que deram mensagem de erro ao enviar)
  • verificar os e-mails antes de tentar enviar, uma lista de e-mails que posssívelmente serão rejeitados ou com domínios fora do comum. Podemos adicionar dominios na caixa de texto logo abaixo e podemos ver os domínios já considerados dentroo da lista do Verificar e-mails
  • definir o intervalo de tempo em que os e-mails serão enviados (em minutos) e caso a opção esteja ativa, o contador só conta a partir do primeiro enviu manual, para que o utilizador possa fazer alterações antes de começar a enviar
  • definir o e-mail para envio de avisos, por exemplo, para avisar que a lista foi terminada
